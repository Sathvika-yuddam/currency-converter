# 💱 Currency Converter

A simple Currency Converter Web Application built using HTML, CSS, and JavaScript.
It allows users to convert an amount from one currency of a country to another using real-time exchange rates fetched from an API.

### 🔗 Live Demo:
https://sathvika-yuddam.github.io/currency-converter/

## 🚀 About the Project

This project is a web-based currency converter that helps users quickly calculate the value of one currency in another.
The application fetches latest exchange rates from an API and performs the conversion dynamically using JavaScript.

The project was built to practice:

API fetching in JavaScript

DOM manipulation

Working with dropdown selections

Building responsive UI with CSS

## ✨ Features

🌍 Convert between different international currencies

🔄 Real-time exchange rate fetching from API

💰 Enter any amount and get instant conversion

🏳️ Currency selection with country flags

📱 Simple and responsive user interface

## 🛠️ Technologies Used

HTML5 – Structure of the application

CSS3 – Styling and layout

JavaScript (Vanilla JS) – Conversion logic and API requests

Currency API – Provides latest exchange rates
